        通过体验程序了解 ele-crud

        wll8小于 1 分钟
        此页内容
        • 安装和运行

        为了快速获得完整体验，我们开发了一个基于 nodejs 的体验程序。

        建议的环境：

        • nodejs v14 版以上

        安装和运行

        # 安装程序
npm i -g ele-crud-cli

# 运行程序
npx ele-crud-cli

        首先运行 npm i -g ele-crud-cli 安装 ele-crud-cli，然后再运行 npx ele-crud-cli 启动程序，这时候你的控制台可以看到以下内容：

        服务已启动, 打开链接 http://127.0.0.1:8800/live/ 即刻使用吧~

        然后访问控制台中出现的链接即可。
